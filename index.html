OneVote

One Person One Vote
Blockchain Based Remote Voting System

Introduction

OneVote is an online voting service provider where clients from different schools, universities, law firms, multinational organizations, award shows, government institutions etc. would be able to experience a fraud free voting process. The main goal of the project is to make the election process secure and transparent with the use of Ethereum Blockchain which exhibits all the properties required for a secure election.

A portable decentralized application through which a voter can easily cast and track his/her casted vote easily and securely which would eventually increase voter confidence and remove the middle man from the whole process.

Immutability

Once a block is created in a blockchain it cannot be changed or tampered. Blockchainâ€™s property of being immutable makes it the best technology to be used in voting system to ensure that that the casted voted cannot be changed once it has been casted.

Voter Confidence

Once you have casted your vote you never know what happened to it. You trust few individuals to count all the casted votes accurately without any external interference. There is no way for any voter to track his/her casted vote and make sure that the vote is casted to the right candidate. But here for every casted vote the voter gets a transaction ID to track and verify their casted vote.

Cost and Efforts

This decentralized application will help in making the whole process automatic without the need of any middle man and there is no need to print paper ballots and count votes which is going to result in cost and efforts reduction.
